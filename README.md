# PNG to Bits

Convert PNG images to a C-array of pixels.

## Usage
```terminal
USAGE:
    png-to-bits [FLAGS] [OPTIONS] <INPUT> <OUTPUT>

FLAGS:
    -c, --compact      8 pixels are packed into one byte.
    -d, --debug        Print debug info.
    -f, --flip-bits    Flip bits. Does nothing unless compact is set. 0b11000000 -> 0b00000011.
    -h, --help         Prints help information
    -i, --invert       Invert all bits. 0xff -> 0x00.
    -p, --print        Print ascii to the terminal.
    -V, --version      Prints version information

OPTIONS:
        --blue <blue>      Blue threshold. Default 127.
        --green <green>    Green threshold. Default 127.
        --red <red>        Red threshold. Default 127.

ARGS:
    <INPUT>     PNG input file.
    <OUTPUT>    Output file
```
